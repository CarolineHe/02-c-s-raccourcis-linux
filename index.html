Raccourcis Environnement Linux

  • Ctrl + Alt + L : Verrouiller la session

  • Ctrl + Alt + T : Ouvrir un terminal

  • Ctrl + Alt + Flèches : Changer de bureau

  • Ctrl + + ou - : Zoomer

  • Ctrl + F : Search depuis la fenêtre

  • Ctrl + Tab : Passer d'un onglet à l'autre dans la même fenêtre

  • Alt + Tab : Changer de fenêtre

  • Alt + Tab (enfoncé) : Afficher les fenêtres et basculer

  • Ctrl + W : Fermer l'onglet actuel

  • Ctrl + S : Sauvegarder

  • Ctrl + N : Ouvrir un nouvel onglet (Atom + Gedit) / une nouvelle fenêtre (Mozilla)

  • Ctrl + T : Nouvel onglet (Mozilla)
